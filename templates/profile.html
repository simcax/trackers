API Usage

Use your API keys to access the Trackers API programmatically. Include your API key in the Authorization header:

Authorization: Bearer your_api_key_here

Available Endpoints

  • • GET /trackers - List your trackers
  • • POST /add_tracker - Create new tracker
  • • GET /tracker-values - Get tracker values
  • • POST /tracker-values - Add tracker values

Security Notes

  • • Keep your API keys secure and private
  • • Set expiration dates for better security
  • • Keys showing "Expires Soon" expire within 7 days
  • • Invalidate unused or compromised keys
  • • API keys only access your own data
